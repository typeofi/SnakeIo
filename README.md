### 蛇蛇大作战设计思路

我们要实现蛇蛇大作战的效果，首先我们分析蛇的运动。蛇可以360度平滑移动，并且从左到右掉头，会有个角度转弯。

要实现上面功能得考虑两点

* 记录当前蛇运动的角度

* 保存蛇的运动轨迹

>蛇的角度记录了当前的运动方向，通过方向计算x和y方向的加速度。
保存当前的运动轨迹 通过轨迹来画蛇身
